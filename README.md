# MTTA
### 국문 초록
기존 Test-time adaptation(TTA)는 close-set 상황을 전제로 data shift가 존재하는 실제 산업의 open-set 문제에서 한계가 있다. 본 연구는 실제 반도체(WM-811K) 데이터의 알려진 결함과 알려지지 않은 결함이 혼재된 현실적 open-set 상황에서의 문제를 다룬다. TTA를 기반으로 한 Multi-bin Test-Time adaptation(MTTA)를 제안하여여 실제 산업 데이터를 직면 했을 때 처음 보는 데이터를 구별하고, 이를 이용해 처음 보는 데이터의 식별 성능을 높이는 기법을 제안하여 정확한 결함 탐지 정확도를 높인다.
### 영문 초록
Existing Test-time adaptation (TTA) has limitations in real-world open-set problems where data shift exists, as it assumes a close-set scenario. This study addresses problems in realistic open-set scenarios where known defects and unknown defects coexist in actual semiconductor (WM-811K) data. We propose Multi-bin Test-Time adaptation (MTTA) based on TTA. This technique distinguishes unseen data when confronted with real industrial data and utilizes this to enhance the identification performance of unseen data, thereby improving accurate defect detection accuracy.
